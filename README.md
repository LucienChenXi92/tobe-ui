## TOBE UI

Live Demo: https://lucienchen.xyz

TOBE is an open-source blog system that contains frontend visualization and backend content management in two parts, it supports online creation, edition, and release content. The UI of TOBE is implemented with React and Material-UI framework.

### How to start the project

You need to download this project to your local and prepare the following basic environment:

Nodejs: v14.18.3 or above

Once your environment is ready, you can run `npm install` to install the necessary packages, and then run `npm start` to boot the project, you can access the page via `localhost:3000`.
